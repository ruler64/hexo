# Docker 原理

镜像和容器

当我们利用 Docker 安装应用时，Docker 会自动搜索并下载应用镜像（image）。镜像不仅包含应用本身，还包含应用运行所需要的环境、配置、系统函数库。Docker 会在运行镜像时创建一个隔离环境，称为容器（container）。

镜像仓库：存储和管理镜像的平台，Docker 官方维护了一个公共仓库：Docker Hub。
